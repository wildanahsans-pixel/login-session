# Login Session PHP

Project sederhana login dan dashboard menggunakan PHP session.

## Fitur
- Login menggunakan username dan password
- Session untuk menjaga status login
- Dashboard hanya bisa diakses setelah login
- Logout untuk menghapus session

## Cara Menjalankan
1. Install XAMPP
2. Simpan folder project di htdocs
3. Jalankan Apache
4. Akses:
   http://localhost/login-session/login.php
